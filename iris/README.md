## How to load the data

### Python Pandas

```python
import pandas as pd
iris = pd.read_csv("https://raw.githubusercontent.com/practiceprobs/datasets/main/iris/iris.csv")
```

### R data.table

```r
library(data.table)
iris = fread("https://raw.githubusercontent.com/practiceprobs/datasets/main/iris/iris.csv")
```
